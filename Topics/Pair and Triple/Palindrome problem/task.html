Palindrome problem

A palindrome is a word which reads the same backward as forward, such as madam or racecar. It means, second word is reversed first. Write a function named palindrome with a Pair parameter with two words, which returns a Triple with the two given words and the word "true" if the two words are a palindrome or "false" if they are not. The function should be case-insensitive.


Sample Input:
cat dog

Sample Output:
(cat, dog, false)


Sample Input:
Otto otto

Sample Output:
(Otto, otto, true)



Memory limit: 256 MB
Time limit: 8 seconds
